### 議案  
#### 1. スタッフの選出  
   * 議題  
     + 第19回小金井アクアスロン大会の役員を決定したい  
   * 決定事項  
     + 基本的には第18回大会と同様で進めたい  
     + ランパートは、橋爪さんがランのサブチーフとして追加し3名体制とする  
     + 競技説明は浅利さんに加え、審判員資格保持者(高橋、鳥野見、綿貫)が関わることを検討  
       (JTU公式ルールに準拠する大会と考える。審判員ドレスコードなどは別途検討）  
     + 集計は、金久保さんのご意向を踏まえて湯山が調整  
     + フォトグラファーは谷口さんで調整

#### 3. 選手募集ポスター  
   * 議題  
     + 原案を確認いただき、来週の印刷を行いたい  
       （文言や協賛関係で指摘を頂きたい）  
     + 協賛各社の名前を印字するので、決定した協賛企業を教えてほしい  
   * 決定事項  
     + 今年は対応が遅れているため、協賛(予定)として昨年の各社を入れ発注する  
       (昨年の実績だと、この時期には決定していたが、各社が遅れている状況)  

#### 4. 選手募集要項  
   * 議題  
     + 本年度の大会要項の読み合わせ原稿の確認。  
       https://github.com/KoganeiTriathlon/KoganeiAquathlon/blob/master/EntryRequirements/2017_Leaflet.md  
     + 誓約書（事故などの取り扱い、リザルト掲載、写真など）の記述について  
     + 協賛などの記述について  
   * 決定事項  
     + 各自で確認していただいて、5/17(水)までに綿貫へ連絡。その時点で発注する。  
     + 個人情報保護の時流で年齢を非表示にする大会が増えてきたが、年齢格差を越えた競技の観点から復活する流れでもある。  
       （誓約書で、リザルトに掲載する情報を承諾してもらう対応を行う）  
     + 選手募集ポスターと同様、協賛(予定)として対応する  

#### 5. 大会要項（冊子）  
   * 議題  
     + 今年度大会の読み合わせ原稿の確認  
     https://github.com/KoganeiTriathlon/KoganeiAquathlon/blob/master/CompetitionGuidelines/2017_Booklet.md   
   * 決定事項  
     + 各自で読み合わせ原稿を確認し、指摘があれば7月中旬までに綿貫へ伝える  

#### 6. コミュニケーションツール  
   * 議題  
     + Facebook Messengerだけに頼っているが、スレッド分けできず情報のキャッチアップに困っている  

   * 決定事項  
     + とりあえず、小金井市トライアスロン連合と小金井アクアスロン大会を分離し様子を見る  

#### 7. 連合のぼり旗作成  
   * 議題  
     + 作成するデザイン案の絞り込み  
   * 決定事項  
     + A案、B案いずれも捨てがたい。よって両方作る  
     + 作成する会社は、[のぼりキング](http://www.i-nobori.com/c_page/5975/0001.html "リンク先を参照する。")  
     + 上記店舗の単価だと14本発注できる。A案を7本・B案を7本注文すること(税込み3,672円×14本)  
     + 新島トライアスロンに間に合うようすぐにでも発注すること  

#### 8. ステッカー作成  
   * 議題  
     + ステッカーのニーズ有無の確認  
   * 決定事項  
     + サポートカーにつけるものが欲しい  
     + ヘルメットやフレームに貼る小さいのは、サイズや数量を綿貫が集計中  
       (希望者は連絡すること)
